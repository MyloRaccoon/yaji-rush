Yaji-Rush ver 1.0.2

pour démarer le jeu :
	- Installer Python
	- Installer Pygame (pour ce faire, taper 'pip install pygame' dans le cmd)
	- Lancer le fichier 'main.py'

dans le jeu : Les flèches, entrer, et échape sont utiliser pour naviguer dans les menus.
